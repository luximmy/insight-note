## 1. 项目定位与核心价值

- **名称**：InsightNote —— AI 智能文档/研报分析助手。
- **核心卖点**：Generative UI + **真 RAG**（检索增强，非“全文塞进上下文”冒充）。
- **面试亮点**：AI 通过 Tool 触发可交互 React 组件；Next.js App Router / 流式与数据流讲清楚。

## 2. 当前技术栈（以仓库为准）

- Frontend: Next.js (App Router), Tailwind CSS, Shadcn UI, Framer Motion（按需）。
- AI: Vercel AI SDK, OpenAI（或兼容 Provider）。
- Data: RAG = PDF → 分块 → Embedding → 向量库 → top-k 检索 → 注入对话。

## 3. MVP 摘要（详细见 `docs/prd.md`）

- **北极星**：5 分钟内上传 1 份 PDF → 对话获 **基于文档** 的回答 → 至少 **1 张图**（生成式 UI）。
- **表格类 Generative UI**：V1.1，不阻塞 MVP。
- **验收**：Grounding、边界拒答、图表工具端到端可演示。

## 4. 进度快照
